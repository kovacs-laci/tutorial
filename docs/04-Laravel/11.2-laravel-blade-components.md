# Blade komponensek

A Blade komponensek modern, tiszta megoldást adnak az újrahasznosítható elemek kezelésére.  
Az Artisan parancs automatikusan létrehozza a komponens **osztályát** és a hozzá tartozó **Blade sablont**.

👉 Parancs:
```bash
php artisan make:component Név
```

Ez két fájlt hoz létre:
- `app/View/Components/Név.php` → a komponens osztálya
- `resources/views/components/név.blade.php` → a komponens sablonja

Az osztály **nem üres**, hanem tartalmaz egy alap konstruktor és egy `render()` metódust:

```php
class Flash extends Component
{
    public function __construct()
    {
        // Ide kerülhetnek a konstruktorban átadott paraméterek
    }

    public function render(): \Illuminate\View\View|\Closure|string
    {
        return view('components.flash');
    }
}
```

---

## 1. Flash komponens

**Parancs:**
```bash
php artisan make:component Flash
```

**Sablon:**
`resources/views/components/flash.blade.php`
```bladehtml
@if (session('success'))
    <div class="success">{{ session('success') }}</div>
@endif

@if (session('error'))
    <div class="error">{{ session('error') }}</div>
@endif

@if ($errors->any())
    <div class="error">
        <ul>
            @foreach ($errors->all() as $error)
                <li>{{ $error }}</li>
            @endforeach
        </ul>
    </div>
@endif
```

**Használat:**
```bladehtml
<x-flash />
```

## 2. Toolbar komponens

**Parancs:**
```bash
php artisan make:component Toolbar
```

**Osztály:**
`app/View/Components/Toolbar.php`
```php
class Toolbar extends Component
{
    public bool $isAuthenticated;
    public array $routes;

    public function __construct(bool $isAuthenticated, array $routes)
    {
        $this->isAuthenticated = $isAuthenticated;
        $this->routes = $routes;
    }

    public function render(): \Illuminate\View\View|\Closure|string
    {
        return view('components.toolbar');
    }
}
```

**Sablon:**
`resources/views/components/toolbar.blade.php`
```bladehtml
<div class="toolbar">
    @if($isAuthenticated)
        <a href="{{ $routes['create'] ?? '#' }}">{{ __('Új') }}</a>
    @endif
    <a href="{{ $routes['csv'] ?? '#' }}">{{ __('CSV') }}</a>
    <a href="{{ $routes['pdf'] ?? '#' }}">{{ __('PDF') }}</a>
    <a href="{{ $routes['mail'] ?? '#' }}">{{ __('Mail') }}</a>
</div>
```

**Használat:**
```bladehtml
<x-toolbar :isAuthenticated="$isAuthenticated" :routes="$routes" />
```

## 4. SearchForm komponens

**Parancs:**
```bash
php artisan make:component SearchForm
```

**Osztály:**
`app/View/Components/SearchForm.php`
```php
class SearchForm extends Component
{
    public string $action;

    public function __construct(string $action)
    {
        $this->action = $action;
    }

    public function render(): \Illuminate\View\View|\Closure|string
    {
        return view('components.search-form');
    }
}
```

**Sablon:**
`resources/views/components/search-form.blade.php`
```bladehtml
<div class="search-bar">
    <form method="get" action="{{ $action }}" accept-charset="UTF-8" class="search-form">
        <input type="search" name="needle" value="{{ request('needle') }}" placeholder="{{ __('Keresés') }}">
        <button type="submit" title="{{ __('Keres') }}">
            <i class="fa fa-search"></i>
        </button>
    </form>
</div>
```

**Használat:**
```bladehtml
<x-search-form :action="route('makers.index')" />
```

## 5. Példa az Index nézetben

`resources/views/makers/index.blade.php`

```bladehtml
@extends('layouts.app')

@section('title', __('Gyártók listája'))

@section('content')
    <h1>{{ __('Gyártók') }}</h1>

    <x-search-form :action="route('makers.index')" />

    <x-toolbar :isAuthenticated="$isAuthenticated" :routes="[
        'create' => route('makers.create'),
        'csv' => route('makers.export.csv'),
        'pdf' => route('makers.export.pdf'),
        'mail' => route('makers.mail'),
    ]" />

    <x-flash />

    <table>
        <thead>
        <tr>
            <th>#</th>
            <th>{{ __('Név') }}</th>
            <th>{{ __('Műveletek') }}</th>
        </tr>
        </thead>
        <tbody>
        @forelse($entities as $maker)
            <tr>
                <td>{{ $maker->id }}</td>
                <td>{{ $maker->name }}</td>
                <td>
                    <a href="{{ route('makers.show', $maker->id) }}">{{ __('Megtekintés') }}</a>
                    @if($isAuthenticated)
                        <a href="{{ route('makers.edit', $maker->id) }}">{{ __('Szerkesztés') }}</a>
                        <form action="{{ route('makers.destroy', $maker->id) }}" method="POST">
                            @csrf
                            @method('DELETE')
                            <button type="submit">{{ __('Törlés') }}</button>
                        </form>
                    @endif
                </td>
            </tr>
        @empty
            <tr>
                <td colspan="3">{{ __('Nincs gyártó a rendszerben.') }}</td>
            </tr>
        @endforelse
        </tbody>
    </table>

    <x-footer />
@endsection
```

## 6. Összegzés

- A komponensek **osztályból** és **sablonból** állnak.
- Az osztály **nem üres**, hanem tartalmaz egy konstruktor és egy `render()` metódust.
- A konstruktorban definiáljuk a paramétereket, amelyeket a sablonban közvetlenül használhatunk.
- A sablon a `resources/views/components` mappában él.
- A használat `<x-név />` formában történik, paraméterek átadásával.
- Példák: `<x-flash />`, `<x-toolbar />`, `<x-footer />`, `<x-search-form />`.
