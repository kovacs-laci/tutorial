# Toolbar műveletek – PDF, CSV, Email

A Laravelben gyakori igény, hogy a listázó oldalakon a **toolbar** gombjaihoz különféle export vagy értesítési funkciókat kapcsoljunk.  

Ebben a fejezetben három tipikus példát mutatunk be a **County–City** párosra:

- PDF export
- CSV export
- Email küldés

---

# 1. PDF export

## Telepítés

A PDF generáláshoz a következő csomagot használjuk:

```

composer require barryvdh/laravel-dompdf

````

Dokumentáció:  
https://github.com/barryvdh/laravel-dompdf

---

## Route

`routes/web.php`

```php
Route::get('/counties/export/pdf', [CountyController::class, 'exportPdf'])
    ->name('counties.export.pdf');
````

---

## Controller

`app/Http/Controllers/CountyController.php`

```php
use Barryvdh\DomPDF\Facade\Pdf;

public function exportPdf(Request $request)
{
    $counties = County::orderBy('name')->get();

    $pdf = Pdf::loadView('pdf.counties', [
        'entities' => $counties
    ]);

    return $pdf->download('counties.pdf');
}
```

# 2. CSV export

## Route

```php
Route::get('/counties/export/csv', [CountyController::class, 'exportCsv'])
    ->name('counties.export.csv');
```

---

## Controller

```php
public function exportCsv(Request $request)
{
    $needle = $request->get('needle');

    try {

        $query = County::query();

        if ($needle) {
            $query->where('name','like',"%{$needle}%");
        }

        $counties = $query
            ->orderBy('name')
            ->get();

        $csv = "ID,Név\n";

        foreach ($counties as $county) {
            $csv .= "{$county->id},{$county->name}\n";
        }

        return response($csv)
            ->header('Content-Type','text/csv')
            ->header(
                'Content-Disposition',
                'attachment; filename="counties.csv"'
            );

    } catch (\Exception $e) {

        return redirect()
            ->route('counties.index')
            ->with(
                'error',
                "Nem sikerült exportálni a vármegyéket: ".$e->getMessage()
            );
    }
}
```

# 3. Email küldés

## Előkészítés

Dokumentáció:
[https://laravel.com/docs/12.x/mail](https://laravel.com/docs/12.x/mail)

Teszteléshez használható a **MailHog**:

[https://github.com/mailhog/MailHog/releases](https://github.com/mailhog/MailHog/releases)

A MailHog egy lokális SMTP szerver, amely elkapja a leveleket, és böngészőben jeleníti meg:

```
http://localhost:8025
```

Így a levelek **nem kerülnek valódi címekre**.

---

## `.env` konfiguráció

```
MAIL_MAILER=smtp
MAIL_HOST=127.0.0.1
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="info@example.com"
MAIL_FROM_NAME="${APP_NAME}"
```

---

## Artisan parancsok

```
php artisan make:mail CountiesMail
php artisan make:view mail.counties
```

---

---

## Route

```php
Route::get('/counties/mail', [CountyController::class, 'sendMail'])
    ->name('counties.mail');
```

---

## Controller

```php
use Illuminate\Support\Facades\Mail;
use App\Mail\CountiesMail;

public function sendMail()
{
    $counties = County::orderBy('name')->get();

    Mail::to('admin@example.com')
        ->send(new CountiesMail($counties));

    return redirect()
        ->route('counties.index')
        ->with('success','Email elküldve!');
}
```

# 4. Toolbar integráció

A toolbar résznézetben a gombokhoz a megfelelő route-okat rendelhetjük.

```
@include('layouts.toolbar', [

'routes' => [

'csv' => route('counties.export.csv'),
'pdf' => route('counties.export.pdf'),
'mail' => route('counties.mail'),

]

])
```

Ebben a leckében három gyakori toolbar műveletet valósítottunk meg:

**PDF export**

* `barryvdh/laravel-dompdf`

* egyszerű szöveges export

**Email küldés**

* Blade sablon
