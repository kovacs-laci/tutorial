# View-k és Blade fájlok

A **view-k** az MVC architektúra „V” komponensei.  
Feladatuk, hogy a controller által átadott adatokat megjelenítsék a felhasználónak.

Laravelben a view-kat a **Blade sablonmotor** segítségével készítjük.

👉 Dokumentáció: https://laravel.com/docs/12.x/blade

---

## 1. A `resources/views` mappa szerkezete

A view fájlok a következő helyen találhatók:

```
resources/views
```

Ajánlott struktúra counties–cities példával:

```
resources/views/
│
├── layouts/
│   └── app.blade.php
│
├── counties/
│   ├── index.blade.php
│   ├── show.blade.php
│   ├── create.blade.php
│   └── edit.blade.php
│
└── cities/
    ├── index.blade.php
    ├── show.blade.php
    ├── create.blade.php
    └── edit.blade.php
```

Ez a struktúra a modellekhez igazodik.

---

## 2. View létrehozása Artisan paranccsal

Új view fájl létrehozása:

```bash
php artisan make:view counties.index
```

Ez létrehozza:

```
resources/views/counties/index.blade.php
```

> Ha a `make:view` parancs nem elérhető, a fájl manuálisan is létrehozható.

---

## 3. View visszaadása a Controllerből

Példa:

```php
public function index()
{
    $counties = County::all();

    return view('counties.index', compact('counties'));
}
```

A `view()` függvény első paramétere:

```
mappa.fájlnév
```

A második paraméter az adatok átadása.

## 5. Layout használata

A layout egy központi sablon, amely tartalmazza:

* HTML szerkezet
* navigáció
* lábléc
* közös elemek
* és egy tartalom helyet

### Layout fájl:

```
resources/views/layouts/app.blade.php
```

### Tartalom helye a layoutban:

<BladeTabs
plain={
`@yield('content')`
}
bootstrap={
`<main class="container py-4">
@yield('content')
</main>`
  }
  tailwind={
`<main class="max-w-4xl mx-auto p-4">
  @yield('content')
</main>`
  }
/>

---

### Oldal a layout használatával

<BladeTabs

## 7. Keresés a listázó oldalon

### Controller (counties):

```php
public function index(Request $request)
{
    $query = County::query();

    if ($request->filled('needle')) {
        $query->where('name', 'like', '%' . $request->needle . '%');
    }

    $counties = $query
        ->orderBy('name')
        ->paginate(10);

    return view('counties.index', compact('counties'));
}
```

## 10. Összefoglalás

* A view-k a `resources/views` mappában találhatók.
* A `php artisan make:view` paranccsal hozhatók létre.
* A controller a `view()` függvénnyel adja vissza őket.
* A Blade sablonmotor kezeli a megjelenítést.
* A layout biztosítja az oldal szerkezetét.
* A résznézetek újrahasznosítható elemek.
* A keresés `Request` alapján történik.
* A lapozás a `paginate()` metódussal működik.
* A linkekben ajánlott named route-ot használni.
