# Felhasználókezelés és autentikáció Laravelben

A legtöbb webalkalmazásban szükség van arra, hogy bizonyos műveleteket csak **bejelentkezett felhasználók** végezhessenek el.

Laravel beépített autentikációs megoldásokat kínál.  
Ebben a leckében a **Laravel Breeze** csomagot használjuk, amely egyszerű és könnyen tanulható autentikációs rendszert biztosít.

Dokumentáció:  
https://laravel.com/docs/12.x/starter-kits

---

# 1. Laravel Breeze telepítése

Telepítés fejlesztői csomagként:

```bash
composer require laravel/breeze --dev
````

Scaffold generálása:

```bash
php artisan breeze:install
```

Ez létrehozza:

* login oldal
* regisztrációs oldal
* jelszó reset funkció
* autentikációs route-okat
* controller osztályokat
* Blade view-kat

---

# 2. Adatbázis migráció

A felhasználók tárolásához futtassuk a migrációkat:

```bash
php artisan migrate
```

Ez létrehozza a `users` táblát.

---

# 3. Frontend build

Breeze a frontend asseteket **Vite segítségével** kezeli.

```bash
npm install
npm run build
```

Fejlesztés közben használható:

```bash
npm run dev
```

---

# 4. Autentikációs route-ok

A Breeze automatikusan létrehozza a következő route-okat:

| route              | funkció       |
| ------------------ | ------------- |
| `/login`           | bejelentkezés |
| `/register`        | regisztráció  |
| `/logout`          | kijelentkezés |
| `/forgot-password` | jelszó reset  |

Ezeket a `routes/auth.php` fájl tartalmazza.

---

# 5. Bejelentkezett felhasználó lekérdezése

A jelenlegi felhasználó elérhető az `Auth` osztály segítségével.

```php
use Illuminate\Support\Facades\Auth;

$user = Auth::user();
```

Blade sablonban:

```bladehtml
{{ auth()->user()->name }}
```

---

# 6. Middleware használata

Az autentikációt a `auth` middleware segítségével védhetjük.

### Route védelem

`routes/web.php`

```php
Route::resource('counties', CountyController::class)
    ->except(['index','show'])
    ->middleware('auth');

Route::resource('cities', CityController::class)
    ->except(['index','show'])
    ->middleware('auth');
```

Ez azt jelenti:

| művelet | hozzáférés          |
| ------- | ------------------- |
| index   | bárki               |
| show    | bárki               |
| create  | csak bejelentkezett |
| store   | csak bejelentkezett |
| edit    | csak bejelentkezett |
| update  | csak bejelentkezett |
| destroy | csak bejelentkezett |

---

# 7. Blade autentikációs direktívák

Blade sablonokban egyszerűen ellenőrizhető a felhasználó státusza.

### Bejelentkezett felhasználó

```bladehtml
@auth
<p>{{ __('Bejelentkezve mint:') }} {{ auth()->user()->name }}</p>
@endauth
```

### Vendég felhasználó

```bladehtml
@guest
<a href="{{ route('login') }}">
{{ __('Bejelentkezés') }}
</a>
@endguest
```

---

# 8. Navigáció frissítése

`resources/views/layouts/navigation.blade.php`

```bladehtml
<nav>

<a href="{{ route('counties.index') }}">
{{ __('Vármegyék') }}
</a>

<a href="{{ route('cities.index') }}">
{{ __('Városok') }}
</a>

@auth

<form method="POST" action="{{ route('logout') }}">
@csrf

<button type="submit">
{{ __('Kijelentkezés') }} ({{ auth()->user()->name }})
</button>

</form>

@else

<a href="{{ route('login') }}">
{{ __('Bejelentkezés') }}
</a>

<a href="{{ route('register') }}">
{{ __('Regisztráció') }}
</a>

@endauth

</nav>
```

# 10. Controller védelem

Middleware a controller konstruktorában is használható.

```php
public function __construct()
{
    $this->middleware('auth')->except(['index','show']);
}
```

Ez minden metódust védeni fog az `index` és `show` kivételével.

---

# 11. Autentikáció tesztelése

Laravel tesztekben a `actingAs()` metódus használható.

```php
public function test_authenticated_user_can_create_county()
{
    $user = User::factory()->create();

    $this->actingAs($user);

    $response = $this->post(route('counties.store'), [
        'name' => 'Test County'
    ]);

    $this->assertDatabaseHas('counties', [
        'name' => 'Test County'
    ]);
}
```

Vendég felhasználó teszt:

```php
public function test_guest_cannot_create_county()
{
    $response = $this->post(route('counties.store'), [
        'name' => 'Test County'
    ]);

    $response->assertRedirect(route('login'));
}
```

# 12. Összegzés

Ebben a leckében megismerkedtünk:

* a **Laravel Breeze** autentikációs rendszerrel
* a **felhasználók adatbázisban tárolásával**
* az **auth middleware** használatával
* a **Blade autentikációs direktívákkal**
* a CRUD műveletek **hozzáférés-szabályozásával**

Az autentikáció biztosítja, hogy az alkalmazásban bizonyos műveletek csak **azonosított felhasználók számára legyenek elérhetők**.
