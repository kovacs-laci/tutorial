# Bevezető

Ez az oldal rövid áttekintést ad arról, hogy milyen témákat találsz ebben a tananyagban.  
A cél, hogy lépésről lépésre, gyakorlati példákkal ismerkedj meg a modern szoftverfejlesztés alapjaival.

---

## Témák

- **Git**  
  Verziókövetés alapjai, tipikus workflow-k, csapatmunka és open source hozzájárulás.

- **PHP**  
  Nyelvi alapok, tipikus minták, és gyakorlati példák webfejlesztéshez.

- **MySQL**  
  Adatbázis‑kezelés, táblák létrehozása, lekérdezések, kapcsolatok és optimalizálás.

- **Laravel**  
  A legnépszerűbb PHP keretrendszer: routing, modellek, migrációk, és teljes alkalmazásfejlesztési workflow.

- **REST API**  
  REST API alkalmazások fejlesztése: PHP, Laravel

- **REST API kliensek**  
  REST API kliensek fejlesztése: PHP, Laravel, HTML+JavaScrip, Flutter

- **Stripe**  
  Online fizetési lehetőség integrálása Laravel projektbe.
---

## Cél

- Egyszerű, érthető példákon keresztül bemutatni a legfontosabb parancsokat és eszközöket.
- Segíteni abban, hogy **önállóan is magabiztosan** tudj dolgozni verziókövetéssel, backend fejlesztéssel, adatbázisokkal és keretrendszerrel.
- Lépésről lépésre építeni a tudást, hogy a kezdő szinttől eljuss a haladó gyakorlati megoldásokig.

---

## Hogyan használd

- A bal oldali menüben a témák logikus sorrendben követik egymást.
- Minden fejezet rövid magyarázatokat és példákat tartalmaz.
- A workflow részek segítenek megérteni, hogyan illeszkednek össze az egyes parancsok és eszközök a mindennapi fejlesztésben.

---

## Következő lépések

- Kezdd a **Git gyorstalpalóval**, hogy biztos alapokra építs.
- Folytasd a **PHP alapokkal**, majd tanuld meg az **MySQL** adatbázis‑kezelést.
- Végül lépj tovább a **Laravel** fejezetekre, ahol mindezek összeérnek egy teljes alkalmazásban.
